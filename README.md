# Sistem Antrian Monitoring MPP Salatiga

Aplikasi monitoring antrian untuk Mal Pelayanan Publik Kota Salatiga.

## Tentang Project

Ini adalah proyek pengembangan yang dibuat secara interaktif dengan bantuan **OpenClaw AI Assistant** via Telegram. Semua fitur dan kode di sini di develop secara langsung melalui chat, tanpa editor manual.

✅ Dibangun dengan Laravel 11
✅ Realtime monitoring antrian
✅ Manajemen user dan hak akses
✅ Multi role (Admin, Operator, Viewer)
✅ Responsif semua perangkat

---

> ⚙️ Project ini di develop seluruhnya menggunakan OpenClaw AI sebagai coding assistant, semua komunikasi dan pembuatan fitur dilakukan lewat Telegram Chat.
